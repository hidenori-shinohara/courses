\begin{exer}{(Exercise 8)}
  Let $pa + qb = 1$ for some $p, q \in \mathbb{Z}$.
  Then $1 \otimes 1 = (pa + qb) \otimes (pa + qb) = pa \otimes pa + pa \otimes qb + qb \otimes pa + qb \otimes qb = 0 + 0 + 0 + 0 = 0$.
\end{exer}
